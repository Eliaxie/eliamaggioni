Progetto: NAS cloud web server

In questo progetto ho utilizzato una Raspberry Pi connessa ad un hard disk per creare un cloud storage online da cui potessi accedere da remoto.
Per il database ho usato MariaDB mentre come gestore del cloud ho utilizzato ownCloud. Ho infine usato un dynamic-DNS per poter raggiungere il cloud storage dall'esterno della mia rete locale. Ho poi configurato il cloud per essere accessibile dall'applicazione mobile, web client e FTP

Progetto: Monitoraggio mezzi pubblici

In questo progetto ho utilizzato una Raspberry Pi connessa ad uno schermo LCD per ricreare un tabellone di arrivi e partenze. Ho modificato del codice open source e connesso con il sito di ATM Milano per la gestione dei mezzi di trasporto.
